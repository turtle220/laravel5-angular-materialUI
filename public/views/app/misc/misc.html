Debugbar

Debugbar allows you to easily debug your app.

You can debug pretty much everything: queries, routes, messages, mails, views, etc..

Cross Origin Resource Sharing (CORS)

Laravel-cors works out of the box. All API endpoints return the allow-access-origin header. The cors middleware is enabled in the global middlewares in app\Http\kernel.php.

Angular Loading Bar

Angular Loading Bar is that loading bar that you see on top when you navigate between routes.

Angular Filters

Laravel5-angular-material-starter comes with some useful angular filters:

  • Capitalize
  • humanReadable
  • trust_html
  • ucfirst
